## ¿Cómo levantar el proyecto localmente?
1. Clonar repositorio.
2. En la carpeta raiz del proyecto, ejecutar el comando `npm install` .
3. En la carpeta raiz del proyecto, ejecutar el comando `npm start`.# freakworld
